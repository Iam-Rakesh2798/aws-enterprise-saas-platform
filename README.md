# AWS Enterprise SaaS Platform

## Project Overview

This project demonstrates the design and implementation of a production-style Employee Management SaaS Platform on AWS.

The objective is to build a scalable, secure, and highly available cloud infrastructure using AWS best practices and Infrastructure as Code.

## Features

- Employee Login
- Employee Management
- Profile Image Upload
- Employee Search
- Reports Dashboard
- Report Downloads

## Technology Stack

- AWS VPC
- Amazon EC2
- Amazon RDS
- Amazon S3
- IAM
- Route 53
- CloudFront
- Application Load Balancer
- Docker
- Terraform
- GitHub Actions
- CloudWatch
